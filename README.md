# ✅ To-Do List App

A simple and clean **To-Do List** made using **HTML, CSS, and JavaScript**. You can add tasks, mark them as done, and delete them.

## 📝 Features

- Add a task
- Mark task as completed
- Delete a task
- Stores data in browser (localStorage)

## 🛠️ Tech Used

- HTML5
- CSS3
- JavaScript 
- localStorage
